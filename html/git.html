Git

  • git init - расшифровывается как "initial-scale", что переводится как "перечислить все файлы в папке" или "вывести на экран список файлов в папке".
  • git add - расшифровывается как "address", делает отслеживаемый файл.
  • git status - расшифровывается как "status",показывает статус файла.
  • git commit - расшифровывается как "commit", операция записи в репозиторий Git
  • git push - вносит изменения на удаленный репозиторий
  • git pull - подтянет изменения из удаленного репозитория в локальный
