Git

  • init - расшифровывается как "initial-scale", что переводится как "перечислить все файлы в папке" или "вывести на экран список файлов в папке".
  • add - расшифровывается как "address", делает отслеживаемый файл.
  • status - расшифровывается как "status",показывает статус файла.
  • commit - расшифровывается как "commit", операция записи в репозиторий Git
  • push - вносит изменения на удоленный репозиторий
  • pull - подтянет изменения из удоленного репозитория в локальный
